# ERC-721 Token

This is a simple NFT token that is deployed to rinkeby testnet 

How to run :

npm install dependencies

npx hardhat compile

DEPLOY
npx hardhat --network rinkeby run scripts/deploy.js

MINT
node scripts/mint.js
